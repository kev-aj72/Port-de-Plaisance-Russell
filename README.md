# Description
Application de gestion des catways et des réservations d’un port de plaisance.

# Installation

1. Installer les dépendances :
npm install

2. Configurer le fichier .env :
URL_MONGO=...
SECRET_KEY=...

3. Lancer le projet :
npm start
